Chat Room

chatroom is a real-time online multi-person chat site

The chat application can accept multiple users to connect to the website and enter the chat, and mainly realizes the following functions

  • Chat app notifies existing users when a new user logs in
  • The chat application will display a list of users currently chatting
  • The chat application notifies the user when a new user joins the chat
  • The chat application notifies the user when the user exits the chat
  • The chat app will notify other users when other connected users disconnect

usage instructions:

  • The backend of the site uses Node.js and the Express framework
  • Use socket.io as the socket of the chat room as the communication method of the chat room, and perform operations after receiving the corresponding method according to the logical binding
